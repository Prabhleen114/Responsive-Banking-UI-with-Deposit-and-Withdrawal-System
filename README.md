# Responsive Banking UI

## Overview

This project is a responsive Banking User Interface built using HTML, CSS, and JavaScript.  
It simulates basic banking operations such as deposits and withdrawals while maintaining real-time balance updates.

The project demonstrates responsive design principles, DOM manipulation, and basic state management in a front-end application.

---

## Key Features

- Dynamic balance display
- Deposit and withdrawal functionality
- Input validation and error handling
- Responsive layout using Flexbox
- Media query-based adaptive design
- Clean and intuitive UI

---

## Functional Capabilities

- Users can deposit money into their account
- Users can withdraw funds with validation
- Prevents negative balance transactions
- Provides real-time UI feedback
- Maintains transaction state dynamically

---

## Technology Stack

- HTML5
- CSS3 (Flexbox + Media Queries)
- JavaScript (DOM Manipulation & Event Handling)

---

## Learning Outcomes

This project demonstrates:

- Responsive UI design principles
- Flexbox layout implementation
- JavaScript event handling
- DOM manipulation
- Basic front-end state management
- Client-side input validation

---

## How to Run

1. Clone the repository
2. Open `index.html` in any modern browser
3. Use Live Server (optional) for development
